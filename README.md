##  Короткий опис проєкту
У цьому проєкті реалізовано приклад двигуна з пропелером, який здатній утримувати своє положення залежно від ПІД-регулятора , призначений для стабілізації кута тангажу одновісного механічного стенда з пропелерною тягою.

## Функціональні можливості
- цифровий PID-регулятор (P + I + D);
- фільтрація сигналу акселерометра (LPF);
- керування ESC стандартним PWM (1000–2000 мкс);
- ESP8266 у режимі Wi-Fi точки доступу;
- веб-інтерфейс:
  - графік `setpoint / pitch`;
  - онлайн-тюнінг `Kp`, `Ki`, `Kd`;
  - інверсія виходу;
  - калібрування нульового положення;
- телеметрія через WebSocket (20 Гц);
- аварійне вимкнення при перевищенні безпечного кута.
